{{% pageinfo %}} 这是一个占位页面，向您展示如何使用此模板站点。 {{% /pageinfo %}}

如果您的项目有 API、配置或其他参考 - 用户需要查找的任何内容，甚至比单个任务更低级别 - 请将其放在（或链接到）此处。您可以通过将它们放在 `static/` 目录中来提供和链接到使用 Doxygen、Javadoc 或其他文档生成工具创建的生成参考文档。在 [添加静态内容](https://docsy.dev/docs/adding-content/content/#adding-static-content) 中了解更多信息。
对于 OpenAPI 参考，Docsy 还提供了 [Swagger UI 布局和短代码](https://www.docsy.dev/docs/adding-content/shortcodes/#swaggerui)，它使用任何 OpenAPI YAML 或 JSON 文件作为源来呈现 [Swagger UI](https://swagger.io/tools/swagger-ui/)。
